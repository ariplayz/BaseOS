BaseOS

A simple, freestanding C++ Operating System for UEFI environments.

About BaseOS

BaseOS is an experimental operating system built from the ground up to run in a pure UEFI environment. It leverages the power of C++ in a freestanding context, providing a high-level API for console I/O and hardware interaction.

Key Features

  • Pure UEFI: Runs directly on UEFI firmware without legacy BIOS dependency.
  • C++ Library: Custom Console library inspired by C# Console.WriteLine.
  • Freestanding: Minimalist design with custom memory management and C++ ABI stubs.
  • Easy Build: Fully automated build and ISO creation scripts.

Getting Started

To try BaseOS, clone the repository and run the build scripts:

./build.sh
./makeimage.sh
./run.sh

Download

You can find the latest ISO builds in the GitHub Actions artifacts.
